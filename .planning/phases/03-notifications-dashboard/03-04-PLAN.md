**4. Add JavaScript section `// === CONFIG PANEL ===`:**

State:
```javascript
let configTab = 'done';
let serverConfig = {}; // populated by config:updated SSE or initial fetch
let configAudio = null;
```

`async function loadConfig()`: `serverConfig = await (await fetch('/config')).json()`

`function openConfig()`: Add 'open' class to config-panel and config-overlay. Call `applyConfigTab()`.

`function closeConfig()`: Remove 'open' class from both.

`function switchConfigTab(tab)`:
- Set configTab = tab
- Update tab button active states
- Call `applyConfigTab()`

`function applyConfigTab()`:
- Read `serverConfig[configTab]` (with defaults: rate '+0%', pitch '+0Hz')
- Set template input value
- Parse rate value: extract number from "+X%" format, set slider value. Display label.
- Parse pitch value: extract number from "+XHz" format, set slider value. Display label.
- Set preview text to template with {project} replaced by "My Project"
- Clear voice list (will re-populate on Generate)
- If serverConfig[configTab].voice, show "Current: {voice}" above voice list

`function updateRateLabel()`: Read slider value, display as "{val}%"

`function updatePitchLabel()`: Read slider value, display as "{val}Hz"

`function formatRate(val)`: Number → string. val >= 0 ? `+${val}%` : `${val}%`

`function formatPitch(val)`: Number → string. val >= 0 ? `+${val}Hz` : `${val}Hz`

`async function saveCurrentConfig()`:
- Get template from input, rate from slider (formatted), pitch from slider (formatted), voice from serverConfig[configTab].voice
- POST to /select: `{ voice, template, type: configTab, rate, pitch }`
- Update local serverConfig on success

`async function generateVoices()`:
- Get text from preview input. If empty, use template with "My Project".
- Disable button, show "Generating..."
- POST to /generate with { text }
- On success, call `loadVoiceSamples()`
- Re-enable button

`async function loadVoiceSamples()`:
- GET /samples
- Render voice-item for each sample in cfg-voices container
- Each voice-item has: Play button, voice short name, full voice name (muted), "Use This" button
- If voice matches serverConfig[configTab].voice, add .selected class
- Play button: create Audio(`/wav/${name}?t=${Date.now()}`), play
- "Use This" button: call `selectVoice(fullVoiceName)`

`async function selectVoice(voice)`:
- Update serverConfig[configTab].voice = voice
- Call `saveCurrentConfig()` — this POSTs to /select which also clears cache
- Update visual selection in voice list
