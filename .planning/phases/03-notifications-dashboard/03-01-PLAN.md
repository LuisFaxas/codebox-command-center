   - In `generateSamples(text, cb)`, accept optional `rate` and `pitch` params: `generateSamples(text, cb, rate, pitch)`. Format edge-tts command with provided rate/pitch or defaults: `--rate="${rate || '+0%'}" --pitch="${pitch || '+0Hz'}"`. Replace the hardcoded `--rate="-5%" --pitch="-10Hz"`.
   - In `generateCached(type, project, cb)`, read rate and pitch from config: `const rate = cfg.rate || '+0%'` and `const pitch = cfg.pitch || '+0Hz'`. Replace hardcoded `--rate="-5%" --pitch="-10Hz"` with `--rate="${rate}" --pitch="${pitch}"`.

2. Create `public/sw.js` — service worker for push reception:
   ```javascript
   self.addEventListener('push', (event) => {
     const data = event.data ? event.data.json() : {};
     const title = data.type === 'question'
       ? 'Claude needs attention'
       : 'Claude is done';
     event.waitUntil(
       self.registration.showNotification(title, {
         body: data.project ? `${data.project} — ${data.type}` : data.type,
         tag: `claude-${data.type}-${data.project}`,
         renotify: true,
         requireInteraction: data.type === 'question',
       })
     );
   });

   self.addEventListener('notificationclick', (event) => {
     event.notification.close();
     event.waitUntil(
       clients.matchAll({ type: 'window' }).then((windowClients) => {
         if (windowClients.length > 0) {
           windowClients[0].focus();
         } else {
           clients.openWindow('/');
         }
       })
     );
   });
   ```
