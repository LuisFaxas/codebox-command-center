    2. Create `sdk-bridge.js` in project root (next to server.js). This is the abstraction layer per D-04 that isolates the rest of the system from the unstable V2 API.

    Exports:
    - `getSdkSessions()` -- calls `listSessions()` from SDK with no dir filter (sees all projects on CodeBox). Returns array of `{ sessionId, summary, lastModified, cwd, firstPrompt, tag, createdAt }`. On error, returns empty array (silent catch per project convention).
    - `getSdkMessages(sessionId, limit = 20, offset = 0)` -- calls `getSessionMessages(sessionId, { limit, offset })`. Extracts text content from message.content array (filter blocks where type === 'text', join with '\n'). Returns array of `{ type, uuid, sessionId, content }`. On error, returns empty array.
    - `sendResponse(sessionId, text)` -- calls `unstable_v2_resumeSession(sessionId, { model: 'claude-sonnet-4-20250514' })`, then `session.send(text)`, then iterates `session.stream()` collecting assistant text blocks. Returns `{ ok: true, response }` or `{ ok: false, error }`. Calls `session.close()` in finally block.

    Helper: `extractTextContent(message)` -- internal function. Takes a raw message object, filters content blocks for type === 'text', returns joined text string.

    Import pattern:
    ```javascript
    import { listSessions, getSessionMessages, unstable_v2_resumeSession } from '@anthropic-ai/claude-agent-sdk';
    ```

    If the SDK exports differ from documented (e.g., named exports not available), adapt the import. The key is to wrap all SDK calls in try/catch and return safe defaults.

    3. Verify SDK loads: `node -e "import('./sdk-bridge.js').then(m => console.log(Object.keys(m)))"`

    ```javascript
    const MIME_TYPES = {
      '.html': 'text/html',
      '.css': 'text/css',
    ```

    3. Add `serveStatic(pathname, res)` function. Takes URL pathname, resolves to `public/` directory. Security: verify resolved path starts with publicDir (prevent directory traversal). Read file synchronously, set Content-Type from MIME_TYPES lookup (default: `application/octet-stream`), set Content-Length, return true on success, false on not-found. Skip paths that are just `/` (let index.html catch-all handle it).

    4. Add SDK proxy routes BEFORE the catch-all `else` block in the route handler:

    - `GET /sdk/sessions` -- async handler. Calls `getSdkSessions()`, returns JSON array. Status 200.
    - `GET /sdk/sessions/:id/messages` -- extract session ID from pathname (regex: `/sdk/sessions/([^/]+)/messages`). Parse `limit` and `offset` from query params (default 20, 0). Calls `getSdkMessages(id, limit, offset)`, returns JSON array. Status 200.
    - `POST /sdk/sessions/:id/send` -- extract session ID from pathname (regex: `/sdk/sessions/([^/]+)/send`). Parse JSON body for `{ text }`. Calls `sendResponse(id, text)`, returns JSON result. Status 200 on success, 500 on error.

    5. Replace the catch-all `else` block to try static file serving first:
    ```javascript
    } else {
      // Try static file first (for .js, .css, etc.)
      if (pathname !== '/' && serveStatic(pathname, res)) return;
      // Fall through to index.html for SPA
      res.writeHead(200, {'Content-Type': 'text/html'});
      res.end(indexHtml);
    }
    ```

    6. The `/sw.js` explicit route can remain (it sets Service-Worker-Allowed header), but static serving will also handle other JS files in public/.

    IMPORTANT: Keep all existing routes unchanged. Only ADD new routes and modify the catch-all.
