    2. **public/modules/state.js** -- Shared state store with pub/sub:
    - `state` object: `{ sessions: new Map(), selectedSessionId: null, config: { done: {}, question: {} }, connected: false }`
    - `subscribe(event, callback)` -- registers listener. Events: 'session:update', 'session:remove', 'session:select', 'config:update', 'connection:change'
    - `dispatch(event, data)` -- calls all registered listeners for that event
    - `updateSession(sessionData)` -- updates sessions Map, dispatches 'session:update'
    - `removeSession(sessionId)` -- deletes from Map, dispatches 'session:remove'. If removed session was selected, set selectedSessionId to null and dispatch 'session:select'
    - `setSelectedSession(sessionId)` -- sets selectedSessionId, dispatches 'session:select' with { sessionId }
    - `setConnected(bool)` -- sets connected, dispatches 'connection:change'
    - `setConfig(configObj)` -- sets config, dispatches 'config:update'
    - `getSession(id)` -- returns session from Map
    - `getAllSessions()` -- returns Array.from(state.sessions.values())

    3. **public/modules/sse.js** -- SSE connection with auto-reconnect:
    - `connectSSE()` -- creates EventSource to `/events`
    - On `session:update` event: parse data, call `updateSession()`
    - On `session:remove` event: parse data, call `removeSession()`
    - On `trigger` event: parse data, dispatch 'trigger' event for toasts/audio
    - On `config:updated` event: parse data, call `setConfig()`
    - On `open`: call `setConnected(true)`, update `.connection-dot` to add `.connected` class, update `.connection-label` to "Connected"
    - On `error`: call `setConnected(false)`, update dot/label to disconnected/"Reconnecting..."
    - EventSource auto-reconnects natively. No manual retry needed.
    - Export `connectSSE`

    4. **public/modules/utils.js** -- Utility functions extracted from current index.html:
    - `escapeHtml(str)` -- escapes &, <, >, ", ' (all 5 entities per Research "Don't Hand-Roll")
    - `formatDuration(ms)` -- formats milliseconds to human string like "5m", "1h 23m", "< 1m"
    - `formatRelativeTime(timestamp)` -- formats timestamp to "just now", "2m ago", "1h ago", etc. Extract logic from current index.html lines ~240-260.
    - `statusColor(status)` -- returns CSS variable name for status: 'done' -> '--status-done', 'attention' -> '--status-attention', 'stale' -> '--status-stale', 'working' -> '--status-working'
    - `statusLabel(status)` -- returns uppercase label: 'done' -> 'DONE', 'attention' -> 'ATTENTION', etc.
    - Export all functions.

    5. **public/modules/app.js** -- Entry point:
    - Import `connectSSE` from `#sse`
    - Import `subscribe` from `#state`
    - On DOMContentLoaded:
      - Fetch `GET /sessions` to load initial sessions, populate state via `updateSession()` for each
      - Fetch `GET /config` to load initial config, call `setConfig()`
      - Call `connectSSE()`
      - Set up sidebar toggle button click handler (toggle `.collapsed` class on `#sidebar`, persist to `localStorage.getItem('sidebar-collapsed')`)
      - Log "Command Center initialized" to console (startup log per project convention)
    - This is the bootstrap module. It will import session/conversation/sidebar/toast/audio modules in later plans. For now, just import what exists.
