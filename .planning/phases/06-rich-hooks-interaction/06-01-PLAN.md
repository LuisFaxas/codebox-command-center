**server.js** — Rewrite the `/trigger` POST handler (lines 159-204) to route by event type:

1. Parse new fields from the body: `const { type: rawType, project: rawProject, sessionId, machine, cwd, timestamp, toolName, toolTarget, questionText, source } = JSON.parse(body);`

2. Add an `extractToolTarget()` helper function near the top of server.js (after the debounce section):
   ```javascript
   function extractToolTarget(toolName, rawTarget) {
     if (!rawTarget) return '';
     // For file paths, show last 2 segments
     if (rawTarget.includes('/') || rawTarget.includes('\\')) {
       const parts = rawTarget.replace(/\\/g, '/').split('/');
       return parts.slice(-2).join('/');
     }
     return rawTarget.substring(0, 80);
   }
   ```

3. Route by type in the handler:
   - **`type === 'tool'`**: Call `handleToolUpdate(sessionId, toolName, extractToolTarget(toolName, toolTarget))` from sessions.js. Do NOT call emit('trigger'), pushToAll(), or generateCached(). Do NOT check isDuplicate() for tool events (they are high-volume state updates, not notifications). Return `{ ok: true, type: 'tool' }`.
   - **`type === 'session-start'`**: Call `upsertSession({ sessionId, project, machine, cwd, type: 'session-start', source })`. Emit `session:update` (handled inside upsertSession). Do NOT trigger notifications. Skip debounce check. Return `{ ok: true, type: 'session-start' }`.
   - **`type === 'done'` or `type === 'question'`**: Keep existing flow (isDuplicate check, upsertSession, emit trigger, pushToAll, generateCached). For question events, pass `questionText` to upsertSession: `upsertSession({ sessionId, project, machine, cwd, type, questionText })`.

4. Add `POST /sessions/:id/dismiss` endpoint (after the /sessions GET handler):
   ```javascript
   } else if (/^\/sessions\/([^/]+)\/dismiss$/.test(pathname) && req.method === 'POST') {
     const id = decodeURIComponent(pathname.match(/^\/sessions\/([^/]+)\/dismiss$/)[1]);
     dismissSession(id);
     res.writeHead(200, {'Content-Type': 'application/json'});
     res.end(JSON.stringify({ ok: true }));
   ```

5. Update the import from sessions.js to include `handleToolUpdate` and `dismissSession`.
