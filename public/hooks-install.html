Hook Installer

Set up Claude Code notification hooks on a remote machine in 3 steps.

Remote machines only. These instructions are for setting up hooks on machines OTHER than CodeBox (e.g., Lenovo, Mac). CodeBox hooks are configured separately and should not be modified using these instructions.
Prerequisite: Node.js must be installed on the target machine. Verify with node --version.
1 Download the hook script

Create the hooks directory and download the notification script.

mkdir -p ~/.claude/hooks
curl -o ~/.claude/hooks/notify-trigger.cjs /hooks/script
2 Configure settings.json

Add the following hooks to ~/.claude/settings.json. Merge with existing settings if the file already exists.

{
  "hooks": {
    "Stop": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "VOICE_NOTIFY_URL= node ~/.claude/hooks/notify-trigger.cjs done",
            "timeout": 5
          }
        ]
      }
    ],
    "Notification": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "VOICE_NOTIFY_URL= node ~/.claude/hooks/notify-trigger.cjs question",
            "timeout": 5
          }
        ]
      }
    ],
    "PostToolUse": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "VOICE_NOTIFY_URL= node ~/.claude/hooks/notify-trigger.cjs tool",
            "timeout": 3
          }
        ]
      }
    ],
    "SessionStart": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "VOICE_NOTIFY_URL= node ~/.claude/hooks/notify-trigger.cjs session-start",
            "timeout": 3
          }
        ]
      }
    ]
  }
}
3 Test the connection

Verify the hook can reach the server. Run this in a terminal:

echo '{"session_id":"test","cwd":"/tmp"}' | VOICE_NOTIFY_URL= node ~/.claude/hooks/notify-trigger.cjs done

1 Download the hook script

Open PowerShell and run the following to create the hooks directory and download the script.

New-Item -ItemType Directory -Force -Path "$env:USERPROFILE\.claude\hooks"
Invoke-WebRequest -Uri "/hooks/script" -OutFile "$env:USERPROFILE\.claude\hooks\notify-trigger.cjs"
2 Configure settings.json

Add the following hooks to %USERPROFILE%\.claude\settings.json. Merge with existing settings if the file already exists.

{
  "hooks": {
    "Stop": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "set VOICE_NOTIFY_URL=&& node %USERPROFILE%\\.claude\\hooks\\notify-trigger.cjs done",
            "timeout": 5
          }
        ]
      }
    ],
    "Notification": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "set VOICE_NOTIFY_URL=&& node %USERPROFILE%\\.claude\\hooks\\notify-trigger.cjs question",
            "timeout": 5
          }
        ]
      }
    ],
    "PostToolUse": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "set VOICE_NOTIFY_URL=&& node %USERPROFILE%\\.claude\\hooks\\notify-trigger.cjs tool",
            "timeout": 3
          }
        ]
      }
    ],
    "SessionStart": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "set VOICE_NOTIFY_URL=&& node %USERPROFILE%\\.claude\\hooks\\notify-trigger.cjs session-start",
            "timeout": 3
          }
        ]
      }
    ]
  }
}
3 Test the connection

Verify the hook can reach the server. Run this in PowerShell:

echo '{"session_id":"test","cwd":"C:\\temp"}' | node "$env:USERPROFILE\.claude\hooks\notify-trigger.cjs" done
